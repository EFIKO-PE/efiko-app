# EFIKO - Aplicación de Estudio de Mercado

![Logo](https://github.com/EFIKO-PE/efiko-next/raw/main/public/app-icon.ico)

EFIKO es una herramienta avanzada para Administradores que permite realizar estudios de mercado estratégicos utilizando Inteligencia Artificial y métodos estadísticos.

## 🚀 Descargar EFIKO

Puedes descargar la última versión estable de EFIKO directamente desde la sección de **[Releases (Lanzamientos)](https://github.com/EFIKO-PE/efiko-app/releases/latest)**.

### Opciones de Descarga:
*   **Instalador de Windows (`EFIKO.Setup.exe`)**: Recomendado para una instalación permanente en tu computadora.
*   **Versión Portable (`EFIKO.exe`)**: Ideal si quieres probar la app sin instalar nada. Solo descargar y ejecutar.

---
**Nota:** Este repositorio se utiliza exclusivamente para la distribución de los binarios oficiales. El código fuente de la plataforma es de propiedad privada de EFIKO Enterprise.
